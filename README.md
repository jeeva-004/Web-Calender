# Wall Calendar Project 🗓️

Hey this is my wall calender project built with React. Its fully responsive and works on both desktop and mobile devices. I made this because i wanted a real looking calender on the web.

## Features

* **3D Page Filp**: when you change the month it filps over like a real physical calender. 
* **Select Dates**: you can select a single day or range of days to add your own notes!
* **Save Notes**: note data is kept safely in the localstorage so you don't lose it if you refresh.
* **Responsive Design**: breaks down to a nice single column on mobile.

## How to run 

just clone the repo and run these commands. make sure you have node installed.

```bash
npm install
npm run dev
```
